# FUT-K privacy notice (LGPD / GDPR)

**FUT-K processes no personal data of its users.**

- No accounts, no cookies, no trackers, no analytics beacons. The optional
  API keys identify *applications*, not people.
- Server logs and metrics contain client IPs transiently for rate limiting
  and operations; they are kept in process memory / standard rotating logs
  and are not profiled or shared.

**About the sports data**: match events reference professional players'
names and in-match actions. This is publicly available sports performance
information published by StatsBomb under their open-data terms, processed
under legitimate interest (LGPD art. 7º IX / GDPR art. 6(1)(f)) for
statistical analysis of public sporting events. FUT-K does not enrich it
with private information. Requests concerning this data should be directed
to the source dataset (StatsBomb Open Data); FUT-K instances only mirror it.

Data controller for a given FUT-K deployment: whoever operates that
deployment. This repository ships no personal data.
