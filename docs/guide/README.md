# FUT-K user guide

Short, non-technical guides to the concepts you'll meet in the app. The default
view already speaks plain language; these pages are for when you switch on
**Analyst mode** and want to know what the numbers really mean.

- [Interpreting Confidence](./confidence.md) — why 62% with high confidence and
  62% with low confidence are opposite advice
- [How Regimes work](./regimes.md) — the six states of a match and why the same
  number means different things in each
- [What Consensus means](./consensus.md) — how multiple sources become one
  diagnosis (and why disagreement is information)
- [Reading the panel](./panel.md) — every element of the match panel, in order
